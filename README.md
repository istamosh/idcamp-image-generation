# IDCamp: Proyek Image Generation

Halo, ini proyek sederhana untuk submission IDCamp Image Generation.

Proyek ini berisi:
- Notebook pipeline dan notebook Streamlit untuk generate gambar.
- Implementasi logic dasar image generation dengan Stable Diffusion.
- Opsi menjalankan Streamlit di localhost, lalu dibagikan lewat ngrok jika perlu.

Ringkasnya, alur kerja proyek:
1. Install dependency pakai uv.
2. Jalankan notebook yang dibutuhkan.
3. Generate gambar dari prompt melalui antarmuka Streamlit.

Terima kasih sudah mampir. Semoga project kecil ini membantu dan mudah dipakai.
